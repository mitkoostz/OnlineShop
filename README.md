# *E-Commerce store with .NET CORE API & Angular*

## **Used Technologies**:
- ASP.NET Core 3.1.1 Web Api
- Angular 10
- Entity Framework Core 3.1
- MSSQL
- Redis
- Stripe Payments
## **Features:**
- Accounts, Roles
- Basket
- Debit card payments
- Admin Panel
- Product Review system
- Caching client & server side
